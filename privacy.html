Политика обработки персональных данных

Действующая редакция для сайта психологической службы «Психология и развитие».

1. Общие положения

1.1. Настоящая Политика обработки персональных данных разработана в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных» и определяет порядок обработки персональных данных оператором сайта психологической службы «Психология и развитие» (контакт для обращений по персональным данным: sar.romanchenko@gmail.com) (далее — «Оператор»).

1.2. Политика действует в отношении всех персональных данных, которые Оператор получает от пользователей сайта juliegordeeva.github.io/SITE-PDC (далее — «Сайт»).

1.3. Использование Сайта означает согласие пользователя с настоящей Политикой.

2. Состав персональных данных

Оператор обрабатывает следующие данные, предоставляемые пользователем при обращении через Сайт: имя, номер телефона, адрес электронной почты, а также иные сведения, которые пользователь добровольно указывает в тексте обращения.

3. Цели обработки

  • обработка заявок и обращений, поступивших через Сайт;
  • связь с пользователем для уточнения запроса и согласования формата работы;
  • направление информационных и рекламных рассылок — при отдельном согласии пользователя.

4. Способ сбора данных

Данные передаются пользователем напрямую на электронную почту Оператора (sar.romanchenko@gmail.com) посредством почтового клиента при отправке письма с заявкой или обращением через Сайт. Сторонние сервисы для приёма и обработки заявок (включая таблицы, CRM и конструкторы форм) Оператором не используются.

5. Передача третьим лицам

Персональные данные не передаются третьим лицам для обработки заявок. Сторонние сервисы веб-аналитики и cookie-файлы для отслеживания поведения пользователей на Сайте не используются. Трансграничная передача персональных данных Оператором не осуществляется. Сайт размещается на инфраструктуре GitHub Pages; обращение пользователя уходит из его почтового клиента напрямую на указанный адрес электронной почты.

6. Срок хранения

Персональные данные хранятся до отзыва согласия пользователем либо до достижения цели обработки.

7. Права пользователя

Пользователь вправе отозвать согласие на обработку персональных данных, запросить информацию об обработке своих данных, а также требовать их уточнения или удаления, направив письмо на адрес sar.romanchenko@gmail.com.
